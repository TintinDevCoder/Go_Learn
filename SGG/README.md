# SGG 目录结构说明

SGG（Structured Go Learning）目录是 Go 语言结构化学习的代码集合，按照基础（basic）和高级（advanced）两个层次组织。本文档详细描述各目录和文件的作用。

## basic/ 目录

basic/ 目录包含 Go 语言基础概念的示例代码，所有文件均为独立的 Go 程序，可直接运行。

### 文件列表

1. **basic_data_type.go**
   - **作用**：演示 Go 语言的基本数据类型，包括整数类型（int, int8, int16, int32, int64）、无符号整数类型（uint, uint8, uint16, uint32, uint64）、浮点数类型（float32, float64）、布尔类型（bool）和字符类型（byte, rune）。
   - **内容**：展示了各种数据类型的声明、初始化和使用，包括类型的大小（通过 unsafe.Sizeof）和范围。

2. **basic_type_toString.go**
   - **作用**：演示基本数据类型与字符串类型之间的相互转换。
   - **内容**：包括使用 `fmt.Sprintf`、`strconv` 包进行转换，以及字符串到其他类型的转换方法。

3. **bian_liang.go**
   - **作用**：讲解变量的声明、初始化、作用域和命名规范。
   - **内容**：包括局部变量、全局变量、变量命名规则、变量作用域等基础概念。

4. **const.go**
   - **作用**：演示常量的定义和使用。
   - **内容**：包括普通常量、枚举常量（iota 关键字）、常量组等用法。

5. **escape_char.go**
   - **作用**：展示 Go 语言中的转义字符及其使用。
   - **内容**：包括常见的转义字符如 `\n`（换行）、`\t`（制表符）、`\\`（反斜杠）等。

6. **jinzhi.go**
   - **作用**：演示不同进制（二进制、八进制、十进制、十六进制）的表示和转换。
   - **内容**：包括各种进制字面量的写法，以及进制之间的转换方法。

7. **operator.go**
   - **作用**：讲解 Go 语言中的各种运算符。
   - **内容**：包括算术运算符、关系运算符、逻辑运算符、位运算符、赋值运算符等，以及运算符的优先级。

8. **value_and_reference_type.go**
   - **作用**：对比值类型和引用类型的区别。
   - **内容**：包括值类型（如 int、float、struct）和引用类型（如 slice、map、channel）的内存模型、传递方式和使用注意事项。

## advanced/ 目录

advanced/ 目录包含 Go 语言高级主题的示例代码，既有独立的 Go 文件，也有按主题组织的子目录。

### 子目录说明

#### 1. arrayAndslice/
   - **作用**：专门讲解数组（Array）和切片（Slice）的相关知识。
   - **包含文件**：
     - `array.go`：数组的声明、初始化和基本操作
     - `slice.go`：切片的创建、扩容、拷贝和常用操作
     - `multiarray.go`：多维数组的使用
     - `stringslice.go`：字符串与切片的相互转换

#### 2. file/
   - **作用**：文件操作相关示例，包括文件的读写、创建、删除等。
   - **包含文件**：各种文件操作的实践代码，展示 Go 标准库中文件处理的功能。

#### 3. func/
   - **作用**：深入讲解函数的高级特性。
   - **包含文件**：
     - `f1.go`：函数的基本定义和调用
     - `f2.go`：匿名函数和闭包
     - `f3.go`：递归函数
     - `f4.go`：函数作为参数和返回值（高阶函数）

#### 4. goroutine_and_channel/
   - **作用**：Go 语言并发编程的核心——goroutine 和 channel 的实践。
   - **包含文件**：goroutine 的创建和同步、channel 的使用、select 语句、并发模式等示例。

#### 5. redis/
   - **作用**：Redis 数据库操作的学习材料。
   - **包含文件**：
     - `redis_commands.md`：详细的 Redis 操作指令手册，涵盖字符串、哈希、列表、集合、有序集合等数据结构的命令和使用方法。

#### 6. reflect/
   - **作用**：Go 语言反射（Reflection）机制的学习。
   - **包含文件**：反射的基本用法、类型检查、值操作、结构体字段和方法调用等。

#### 7. struct/
   - **作用**：结构体（Struct）和相关特性的深入学习。
   - **包含文件**：结构体的定义、嵌套、方法、接口实现、组合等。

#### 8. tcp_socket/
   - **作用**：网络编程基础，TCP 套接字通信示例。
   - **包含文件**：TCP 服务器和客户端的实现，包括连接建立、数据收发、并发处理等。

#### 9. timefunc/
   - **作用**：时间和日期相关函数的用法。
   - **包含文件**：时间获取、格式化、解析、计算等操作。

#### 10. union_test/
   - **作用**：Go 语言测试框架的学习和实践。
   - **包含文件**：单元测试、基准测试、测试辅助函数等。

### 独立 Go 文件说明

以下文件位于 advanced/ 目录根目录，每个文件专注于一个特定的高级主题：

1. **args.go**
   - **作用**：演示命令行参数的获取和处理。
   - **内容**：使用 `os.Args` 和 `flag` 包解析命令行参数，包括基本用法和高级选项。

2. **break.go**
   - **作用**：讲解 `break` 语句在循环和 switch 中的使用。
   - **内容**：包括普通 break、带标签的 break，以及在多层循环中的控制流程。

3. **build-in_func.go**
   - **作用**：介绍 Go 语言的内置函数。
   - **内容**：包括 `len()`、`cap()`、`make()`、`new()`、`append()`、`copy()`、`close()` 等内置函数的用法。

4. **err_handle.go**
   - **作用**：Go 语言错误处理机制的实践。
   - **内容**：包括错误创建（errors.New, fmt.Errorf）、错误检查、自定义错误类型、错误包装等。

5. **for.go**
   - **作用**：详细讲解 `for` 循环的各种形式。
   - **内容**：包括基本 for 循环、range 循环、无限循环、嵌套循环等。

6. **goto.go**
   - **作用**：演示 `goto` 语句的使用（虽然不推荐，但作为语言特性了解）。
   - **内容**：goto 的基本语法、使用场景和注意事项。

7. **json.go**
   - **作用**：JSON 数据的编码和解码操作。
   - **内容**：使用 `encoding/json` 包进行结构体与 JSON 的相互转换，包括标签（tag）的使用。

8. **map.go**
   - **作用**：map 数据结构的深入学习。
   - **内容**：map 的创建、增删改查、遍历、并发安全等。

9. **pointer.go**
   - **作用**：指针（Pointer）的详细讲解。
   - **内容**：指针的基本概念、指针运算、指针与函数、指针与结构体等。

10. **print_and_scan.go**
    - **作用**：输入输出操作，包括格式化打印和扫描。
    - **内容**：`fmt.Printf`、`fmt.Sprintf`、`fmt.Scanf` 等函数的用法。

11. **string.go**
    - **作用**：字符串操作的进阶内容。
    - **内容**：字符串拼接、分割、查找、替换、编码转换等。

12. **switch.go**
    - **作用**：`switch` 语句的各种用法。
    - **内容**：基本 switch、类型 switch、表达式 switch、fallthrough 关键字等。

## 学习建议

1. **基础学习**：建议从 basic/ 目录开始，按照文件顺序学习，掌握 Go 语言的基本概念。
2. **进阶学习**：完成基础后，进入 advanced/ 目录，可以按照主题选择学习顺序。
3. **实践为主**：每个文件都是可运行的示例，建议修改代码、添加功能，加深理解。
4. **参考文档**：结合 Go 官方文档和本目录中的 README.md 文件，构建完整的知识体系。

## 运行方法

```bash
# 运行 basic 目录下的示例
cd SGG/basic
go run basic_data_type.go

# 运行 advanced 目录下的独立文件
cd SGG/advanced
go run args.go

# 运行子目录中的示例
cd SGG/advanced/arrayAndslice
go run array.go
```
